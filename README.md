# Adhith M K - Personal Website

This is my personal academic website showcasing my work in Ecological Informatics.

## Features
- Modern, responsive design
- Academic portfolio
- Blog section
- Dynamic trends analysis
- Research & projects showcase
- Social media integration

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- FastAPI (Backend)

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. For backend functionality, run the FastAPI server on port 8000
